This folder contains a couple of dummy ontologies to be used during testsing. 
These are:

- `n4c.rdf` - [NFDI4Culture Ontology](https://nfdi4culture.de/ontology), version 3.0, originally available under the terms of CC0.
- `gnd_20240806.rdf` - [GND Ontology](https://d-nb.info/standards/elementset/gnd), Version 2024-11-07, originally available under the terms of CC0.
- `met-core.rdf`, `met-annot.rdf`: Two custom ontologies created for this purpose only.
